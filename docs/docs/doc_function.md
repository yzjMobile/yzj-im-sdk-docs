# 客户端功能介绍

* 群组
    * 创建群组
    * 退出群组
    * 解散群组   
* 群组管理
    * 修改群组名称
    * 置顶/取消置顶 群组
    * 消息免打扰
    * 添加/删除成员
    * 管理员设置
        * 仅管理员添加成员
        * 仅管理员修改群组名称
        * 仅管理员@所有人
        * 全员禁言
* 消息
    * 复制
    * 转发
    * 撤回
    * 删除
    * 提及
    * 草稿
    * 已读回执（单人和提及）
    * 消息类型
        * 文本
        * 图片
        * 表情
        * 语音
        * 文件
        * 小视频
        * 自定义消息
        * 系统消息
        * 指令消息
